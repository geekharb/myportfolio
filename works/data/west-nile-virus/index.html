West Nile Virus Analysis

Analysing the incidence of West Nile Virus Analysis in Chicago.

View the project on Github 

Project Synopsis

The project aims to develop a model that helps to predict the occurence of West Nile Virus Analysis (WNV) in the Chicago, Illinois. A two step approach was developed for modelling. First, a regression model was fitted to the training data to predict number of mosquitos. This is followed by a classification model to predict if WNV are present. The resulting findings were used to inform the cost-benefit analysis and offer actionable insights to the Disease And Treatment Agency.
