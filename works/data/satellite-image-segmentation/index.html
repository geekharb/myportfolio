Satellite Image Segmentation

Partitioning satellite image into object segments

View the project on Github 

Project Synopsis

The project seeks to apply a technique in computer vision, semantic segmentation, to satellite imagery. The goal is to divide a satellite image into distinct areas base on the objects, i.e. buldings, woodlands, road, etc, in the image. The model was trained on satellite images and corresponding labelled data on regions in Poland. Future explorations would include more representative training data sets in order to improve model generalizability.
