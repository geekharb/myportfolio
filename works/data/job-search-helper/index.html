Job Search Helper

A job search helper using similarity search to rank jobs which are most relevant to your resume

View the project on Github 

Project Synopsis

Save time in your job search by having a helper find relevant jobs based on your resume content. Performs vectorized similarity search across job descriptions and your resume to find the most relevant jobs. Current implementation uses MPNet V2 for vectorization. This is an ongoing project and more features will be added.
