Generative Urbanism: A Generative Urban Design and Analysis Tool

Project Sypnosis

The project is an attempt at a generative urban design and analysis tool capable of generating design options given an input plot. The fundamental assumption is that urban morphology can be described as a amalgamation of various urban and block typologies. By setting various parameters such as gross floor area (GFA), maximum height, green ratio, etc, we are able to influence the resulting outcome, despite the seeming randomness. The tool is developed with the Mobius modelling environment by the Design Automation Lab at NUS.
